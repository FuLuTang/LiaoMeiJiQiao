# 男女思维差异

1.你是不是真的喜欢我？
2.你是否足够优秀？
3.约会的用心程度

## 1. 你是不是真心喜欢我？

### 男生思维
- 我对你专一、真诚，实话实说。
- 我对你好：信息秒回、嘘寒问暖。
- 我希望你好：给你忠恳的建议，直接帮忙。

### 女生思维
1. 专一
   - 像向全世界宣布我是你的女朋友。
   - 一起出门时，眼睛只停留在我身上。
2. 真诚
   - 只说与我有关的信息，尽量挑好的说。
   - 你如何为了我放弃XXX，又如何不顾一切坚定选择我。
3. 对我好
   - 懂得我的缺点，并且包容我的一切缺点。
   - 欣赏我的优点，并且喜欢我的所有优点。
   - 不计后果地为我付出，只为博我一笑。

## 2. 你是否足够优秀？

### 男生思维
- 有车有房、工作好、收入高。
- 努力上进、知识渊博、情绪稳定。

### 女生思维
1. 有没有钱
   - 看你的生活方式是否精彩。
2. 努力上进
   - 公式：他一个XXX，为了XXX竟然能XXX。
3. 知识渊博
   - 各类话题都略知一二，能表达情绪与感受即可。
4. 情绪稳定
   - 即便女人做错事，你也不发火，能够平静沟通。

---

## 3. 约会的用心程度

### 男生思维
- 精心选择餐厅。
- 准时准点赴约。
- 安排一套完美的约会流程。
- 做她喜欢做的事。

### 女生思维
1. 什么是“好的餐厅”？
   - 在自己打扮漂亮的基础上，环境好、尽量好吃。
2. 什么是“完美的约会流程”？
   - 不要让她陷入尴尬的境地，例如：
     - 路边等车太久很傻。
     - 风大吹乱发型。
     - 天气太热容易脱妆。
     - 熟人太多放不开。
     - 有自己不喜欢的人在场。
     - 完全不会的娱乐项目。
     - 各类运动项目（若她不喜欢或不擅长）。
